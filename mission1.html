by Smart Primer Group

MISSION:
Find the number of candies that will fit inside the box. However, this task isnâ€™t as simple as counting the number of candies in the box. If you do individually count each candy it will take too long. You simply have one quarter and the box and you need to estimate how many of them can fit inside the box.

Hi, I'm your personal tutor Mr.Owl


Key Observation: The width, height and depth of each candy is the same. This means that exactly the same number of candies will fit in each different row.

Step 1: Put one coin at the corner of the box. By marking the edge of the coin, count how many such coins can fit along the length, breadth and height of the box.

Step 2: Count the number of coins that fit along the width of the box. Count the number of coins that fit along the length of the box.

Step 3: Keep count of and stack coins on top of each other until you reach the end of the box.

Step 4: You now have the number of coins that fit along the breadth, height and depth of the box! If you multiply these three together you will have the number of coins in the box in total!
