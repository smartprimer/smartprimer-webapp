Mission 1: Approximate the Candies Challenge

MISSION:
Find the number of candies that will fit inside the box. However, this task isnâ€™t as simple as counting the number of candies in the box. If you do individually count each candy it will take too long. You simply have one quarter and the box and you need to estimate how many of them can fit inside the box.

Oops! Try Again. There seems to be something amiss.

Key Observation: The width, height and depth of each candy is the same. This means that exactly the same number of candies will fit in each different row.

Items Required: One rectangular box, one quarter

Step 1: Put one coin at the corner of the box. By marking the edge of the coin, flip the coin to test how many coins of the same dimension will fit along the length.

Step 2: Count the number of coins that fit along the width of the box. Using the same technique in the previous step, flip the coin to see how many such coins will fit along the length.

Step 3: Use the above technique of flipping the coin to keep track of how many such coins will fit along the height.

Step 4: You now have the number of coins that fit along the breadth, height and depth of the box! If you multiply these three together you will have the number of coins in the box in total! Hint: There is another step after this until your final answer.
